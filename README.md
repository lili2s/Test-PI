
# Controle de Encomendas

## Prérequisitos
Python 3.10.0

Para executar projetos deste repositório:

Clonar Repositório
```sh
$ git clone https://github.com/lili2s/Test-PI.git
$ cd Test-PI
$ python -m venv .
$ .\Scripts\activate
$ pip install flask
$ set FLASK_APP=main.py
$ set FLASK_ENV=development
$ flask run
```
